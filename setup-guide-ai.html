01
WHERE IS MY DATA データはどこに？

このアプリは「買い切りのHTMLファイル1枚」です。インストールもログインもなし。その代わり、データの居場所が普通のアプリと少し違うので、最初にここだけ説明させてください。

📦
レシピのデータ
アプリを開いたブラウザの中（localStorage）に保存されます。サーバーには送られません
🔑
APIキー
同じくブラウザの中だけ。別デバイスには勝手に同期しません
☁️
バックアップ先
自分のGoogleドライブ等にJSONで保存。アプリ側はクラウドを持ちません
🔗
デバイス間で共有
エクスポート→別デバイスでインポート、という手動方式
つまり
データはぜんぶあなたのブラウザの中にあります。だから、
・ ブラウザのキャッシュを消すとレシピも消えます
・ 別のブラウザ／別の端末で開くと、そこは空っぽです
・ その代わり、サービス終了や課金変更でデータが人質になることはありません
大事なレシピは、セクション06の手順で定期的にバックアップを取ってください
02
WHAT YOU NEED 必要なもの

AI版を使うには、Googleアカウントが1つ必要です。

このアプリは、AIの部分にGoogleの「Gemini API」を使っています。APIを使うにはGoogleアカウントでのログインが必須で、そこで発行したAPIキーをアプリに登録する、という仕組みです。

🔑
Gemini APIキー
GoogleアカウントでログインしてGoogle AI Studioで無料発行。次のセクション以降で手順を説明します
☁️
クラウドストレージ
バックアップの保存先。Googleドライブなら15GB無料で、AIと同じアカウントで管理できます
🌐
ブラウザ
Safari / Chrome / Edge どれでもOK。iPhone・Android・PC 全部で動きます
📡
ネット接続
AI読み込みとバックアップ保存の時のみ。普段の閲覧・手入力はオフラインでも動きます
Googleアカウントを持っていない場合
アカウント作成は無料で、数分で終わります。→ accounts.google.com/signup
作成したアカウントで、そのままAPIキー取得とGoogleドライブの両方が使えます。
AI機能を一切使わず手入力だけで使う場合は、APIキーもGoogleアカウントも不要です
03
WHAT YOU CAN DO AI版でできること

AI版は、スタンダード版にスクリーンショット読み込み機能を追加したエディションです。レシピのスクショをそのまま貼るだけで、材料・手順・タイトルを自動入力してくれます。

📸
スクショから読み込み
インスタ、クックパッド、メモ帳の写真など画像なら何でもOK
✏️
手入力もそのまま使える
AIを使わず普通に手入力することも引き続き可能
🔑
自分のAPIキーを使用
Googleの無料枠でほぼ無料。キーはデバイスのみに保存
🔒
完全ローカル動作
レシピはブラウザ内のみに保存。詳しくはセクション01へ
📨
テキストから読み込み
このアプリのシェア機能で書き出したテキストを貼り付けるだけでフォームに自動展開
🌐
Googleの無料機能で動く
AI読み込み = Gemini API、バックアップ保存先 = Googleドライブ等。どちらも自分のGoogleアカウントで無料運用できます
AIによる読み込みには Gemini APIキー（Google製）が必要です。取得は無料で、通常の使い方ではほぼ無料枠内に収まります。次のセクションで手順を説明します。
04
GET YOUR API KEY APIキーを取得する

Google AI StudioでGemini APIキーを発行します。Googleアカウントがあれば今すぐ無料で取得できます。

🔗 aistudio.google.com
1
Google AI Studio を開く
aistudio.google.com にアクセスし、Googleアカウントでログインします
2
「Get API key」をクリック
左メニューまたはトップページの「Get API key」ボタンから進みます
3
「Create API key」で新しいキーを発行
プロジェクトを選択（または新規作成）して「Create API key in new project」をクリック
4
キーをコピーして保管
AIzaSy... から始まる文字列がAPIキーです。コピーして次のステップに進みます
取得したキーはGoogle AI Studioの「Get API key」ページでいつでも再確認できます
05
APP SETUP アプリにAPIキーを設定する

取得したキーをアプリに登録します。キーはあなたのブラウザのみに保存され、外部に送信されることはありません。

1
アプリを開く
ダウンロードした my-recipe-book-with-u-ai.html をブラウザで開きます
2
初回起動時のポップアップ
「AIを使うにはAPIキーが必要です」というダイアログが表示されます。そこにキーを貼り付けて「保存」
3
あとから変更するには「設定」から
画面下の「設定」タブ → 「Gemini APIキー」欄からいつでも変更できます
NOTE
設定済みの場合、設定タブに「✓ APIキー設定済み（...XXXXXX）」と表示されます。末尾6文字のみ表示されるので、別のデバイスで登録したかどうかの確認に使えます。
06
HOW TO USE スクショからレシピを読み込む

APIキーが設定できたら、さっそく使ってみましょう。

1
「＋」ボタンでレシピ追加画面を開く
画面右上の「＋ ADD」をタップ
2
「📸 写真を選ぶ」エリアをタップして画像を選択
インスタのスクショ、レシピサイトの写真などOK。写真一覧から最大2枚まで同時に選択できます
3
「✨ AIにレシピを読み取ってもらう」ボタンをタップ
数秒待つと、タイトル・材料・手順が自動で入力されます
4
内容を確認して「💾 保存する」
AIの読み取りは完璧ではないので、内容を確認・修正してから保存しましょう
読み取り精度は画像のテキストが鮮明なほど上がります。手書きメモよりレシピサイトのスクショが得意です
07
SWAP RECIPES レシピを交換する

このアプリ同士なら、レシピをそっくりそのままやり取りできます。LINEでメッセージを送る感覚で、タイトル・材料・手順・メモ・出典まで、まるごと相手のアプリに届きます。

家族や友人がこのアプリを持っていれば、「うちのカレーのレシピ送るね」が、手打ちなし・写真撮り直しなしでできます。

仕組み
シェアボタンを押すと、レシピが整形されたテキストに変換されます。そのテキストをLINE・メール・メモアプリ等で送り、受け取った人はアプリの追加画面に貼り付けるだけ。サーバーを経由しないので、どこにも履歴が残りません。

📤 送る側

1
送りたいレシピを開く
一覧から送りたいレシピをタップ
2
「📤 シェア」ボタンをタップ
整形済みのレシピテキストが表示されます
3
「📋 テキストをコピー」をタップ
クリップボードにコピーされます。「コピーしたよ！」と出れば成功
4
LINE・メール・メモアプリ等に貼り付けて送信
いつも使っている連絡ツールでそのまま送れます
貼り付ける場所はどこでもOK。LINEでもDiscordでもメールでも、テキストが通ればどこでも

📥 受け取る側

アプリを持っていれば、受け取ったテキストをそのまま自分のレシピ帳に取り込めます。AIキー不要。

1
もらったレシピテキストをコピー
LINEやメールの本文を長押しして全選択 → コピー
2
アプリで「＋ ADD」をタップ
追加画面の一番上に紫色の「📨 テキストから読み込む」エリアがあります
3
紫色エリアにペースト → 「📥 フォームに読み込む」
タイトル・材料・手順・メモ・出典が自動でフォームに展開されます
4
内容を確認して「💾 保存する」
受け取ったまま保存してもいいし、自分好みに調整してから保存してもOK

シェアテキストの見た目

こんな形式でコピーされます。アプリを持っていない人が受け取っても、そのまま読めます。

🍳 大根の辛子漬け ──────────────────── 🥕 材料 ・大根　1/2本 ・塩　大さじ1 ・砂糖　大さじ3 ・辛子粉　大さじ1 👩‍🍳 作り方 1. 大根を拍子木切りにする 2. 塩をまぶして30分おく 3. 水気を絞り、砂糖と辛子粉を混ぜる 4. 保存容器に入れて一晩漬ける 💬 コツ 辛子の風味は一晩おくとマイルドになります 📌 出典：母のレシピ

こんな使い方ができます

👩‍👧
母から娘へ
「実家の味」を口頭じゃなくテキストで受け継ぐ。メモも出典も一緒に届く
👯
友達同士で交換
「この前食べた美味しかったやつ」を、次の日にはお互いのレシピ帳に
🏠
家族で共有
夫婦・親子で同じレシピ帳を育てる。誰かが追加したら送り合うだけ

シェアするのはテキストだけです。レシピに設定した写真は送られません。画像が必要な場合は別途LINE等で送ってください。
08
BACKUP / EXPORT / IMPORT バックアップと引っ越し

レシピが増えてきたら、バックアップを取ってください。別のスマホや新しいPCに引っ越しする時もこの機能を使います。

このアプリ自体はクラウドを持たないので、保存先はあなた自身のクラウドストレージ（Googleドライブ・iCloud Drive・Dropbox など、普段使っているもの）を使います。

保存形式
・ JSONファイル … アプリに読み戻せる引っ越し用データ
・ バックアップHTML … アプリがなくても開ける閲覧専用のレシピ集

📤 エクスポート（書き出す）

今のレシピをファイル1個にまとめて書き出します。

1
設定タブを開く
画面下の「⚙ 設定」をタップ
2
「📤 レシピをエクスポート」をタップ
「データのバックアップ」欄にあります
3
保存先を選ぶ
my-recipe-book-YYYY-MM-DD.json というファイル名で、Googleドライブ等クラウドに保存しておくのがおすすめ
月1回バックアップ、くらいの気持ちでOK。レシピを大量に追加した日も忘れずに

📥 インポート（読み戻す）

書き出したJSONファイルを読み込んで、アプリを元の状態に戻します。別の端末への引っ越しもこの手順です。

1
新しい端末でアプリを開く
引っ越し先の端末で my-recipe-book-with-u-ai.html を開いて、設定タブへ
2
「📥 レシピをインポート」をタップ
ファイル選択ダイアログが開きます
3
JSONファイルを選ぶ
クラウドに保存したJSONを選択。読み込み完了のメッセージが出れば成功
インポートすると既存データは上書きされます。元の端末のレシピと新しいファイルのレシピを合体させたい場合は、先に元の端末でエクスポートしてから、JSONファイルをテキストエディタで編集する必要があります（上級者向け）。

📄 バックアップHTML（見るだけ版）

レシピを「アプリなしでも開ける1枚のHTML」として書き出す機能です。スマホのブラウザで開けば、編集はできないけど、読むだけなら誰でもできます。

👀
閲覧専用
追加・編集はできない。見るだけ
📲
端末を問わない
ブラウザさえあれば開けるので、家族と共有しやすい

設定タブ → 「📄 バックアップHTML（見るだけ版）を保存」から書き出せます。普段使いのバックアップはJSON、共有用はHTML、と使い分けるのがおすすめ。

Googleドライブ
無料15GB。このアプリのAIとも相性がいい
🍎
iCloud Drive
iPhoneユーザーなら無料枠5GBで十分足りる
📦
Dropbox など
普段使っているクラウドで全く問題なし

端末内だけはNG
端末を壊すと一緒に消えるので、必ずクラウドへ
どのクラウドを使っても、保存するのはただのJSONファイルです。特別な設定は何もいりません

10
FAQ よくある質問
Q
APIキーはどこに保存されますか？
あなたのブラウザのlocalStorageにのみ保存されます。外部サーバーには一切送信しません。ただし、AI読み込み時は画像データがGemini APIに送信されます（Googleのプライバシーポリシーに準拠）。
Q
スマートフォンでも使えますか？
はい。Safari / Chrome いずれでも動作します。カメラロールの写真をそのままスクショとして使えます。
Q
読み取りがうまくいきません
①APIキーが正しく設定されているか確認 / ②画像のテキストが鮮明か確認 / ③ネット接続が繋がっているか確認。それでも解決しない場合は設定タブのAPIキーを一度削除して再入力してみてください。
Q
AIを使わずに手入力だけしたい
もちろんOKです。スクショエリアを無視して、タイトル・材料・手順を直接入力して「💾 保存する」をタップすれば普通に使えます。
Q
アプリを持ってない人にレシピをシェアできますか？
送ることはできます。シェア機能で生成されるのは普通のテキストなので、LINE等で送れば相手もそのまま読めます。ただし相手のアプリに取り込めるのはアプリを持っている人だけです。
Q
シェアしたテキストは他の人に見られたりする？
シェア機能はサーバーを一切経由しません。コピーしたテキストをどこに送るかは完全にあなた次第で、アプリ側に履歴も残りません。
Q
ブラウザのキャッシュを消すとどうなりますか？
レシピデータとAPIキーが消えます。大切なレシピは定期的にエクスポート（設定タブ → バックアップ）しておくことをおすすめします。
Q
機種変更するときはどうすればいい？
① 古い端末で「📤 エクスポート」→ JSONをクラウド（Googleドライブ等）に保存 → ② 新しい端末でアプリHTMLを開く → ③ 設定タブから「📥 インポート」でJSONを選ぶ、で引っ越し完了です。APIキーは新しい端末で再登録してください（セキュリティのため一緒には移行されません）。
Q
JSONとバックアップHTMLの違いは？
JSON = アプリに読み戻して編集を再開できる引っ越し用データ。バックアップHTML = アプリなしでもブラウザで開ける閲覧専用ファイル。普段のバックアップはJSON、家族と共有したい時はHTML、と使い分けてください。
Q
クラウドを何も使ってないけど大丈夫？
Googleアカウントがあれば、Googleドライブが無料で15GB使えます（このアプリのAI機能もGoogleアカウントを使うので、同じアカウントで統一すると管理がラクです）。iPhoneなら初期設定でiCloud Driveが有効になっていることが多いです。
